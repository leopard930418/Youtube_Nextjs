# public_html

## Run on localhost

This is a static HTML/CSS/JavaScript site. A local web server is included so files such as `footer.html` can be loaded correctly.

### Windows

Double-click `start-localhost.bat`, then open:

http://localhost:8000/

To use another port, run this from the project folder:

`start-localhost.bat 8080`

### Any terminal with Node.js

Run this from the project folder:

`node localhost-server.js`

Then open:

http://localhost:8000/

To stop the server, press `Ctrl+C` in the terminal window.
